Задача:
Створіть клас User з наступними полями: id, name, email.
Реалізуйте репозиторій UserRepository, який буде зберігати
об'єкти класу User у базі даних. Напишіть сервіс UserService,
який надає методи для додавання нового користувача, отримання
списку всіх користувачів та отримання користувача за його ідентифікатором.
Використовуйте Spring Data JPA для взаємодії з базою даних.


Ваше завдання:

Створіть клас User з відповідними полями та методами доступу до них.
Створіть інтерфейс UserRepository, який наслідує від JpaRepository<User, Long>.
Реалізуйте клас UserService, який має конструктор, що приймає об'єкт типу UserRepository,
та реалізує методи для додавання, отримання списку та отримання користувача за ідентифікатором.
Конфігуруйте Spring таким чином, щоб він автоматично створював біни UserRepository та UserService.
Напишіть юніт-тести для перевірки роботи вашого сервісу.
Пам'ятайте, що вам знадобиться встановлення та налаштування бази даних, яку ви будете
використовувати (наприклад, MySQL або H2) для роботи з Spring Data JPA.

Успіхів у вирішенні задачі!
